Ethereum Go

Deps
====

Ethereum Go makes use of a modified `secp256k1-go` and therefor GMP.

Install
=======

```go get https://github.com/ethereum/go-ethereum```


Command line options
====================

-db will launch the db query tool.

Supported arguments `update` and `get`
